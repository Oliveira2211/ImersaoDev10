# Introdução

Site contendo um banco de palavras e suas respectivas traduções e pronúncia em inglês, espanhol e russo.

O conceito do site surgiu duma estratégia que comecei a aplicar quando iniciei meus estudos do idioma russo, como esse idioma tem sido difícil para mim, sempre que eu via alguma coisa que eu sabia como se diz em russo, eu repetia essa palavra para mim mesmo, não muito tempo depois eu notei que várias dessas palavras eu também sabia em inglês e espanhol e logo essa estratégia evoluiu para repetir também nesses dois idiomas.
